About

Genre

A browser game in which the player ship flies in space and destroys enemies.

Technologies

Built using Phaser - free 2D game framework for making HTML5 games, supporting Canvas rendering. Phaser is developed in ES5 JavaScript

Requirements

Game requires a keyboard to play. It means it is unplayable on mobile devices :c
Resolution is 1000 x 650

Controls

Use LEFT and RIGHT for moving. Press SPACE to fire.
